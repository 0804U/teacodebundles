# HTML

## Improves the HTML coding.

Made by [twitter] @Apptorium

### Tag

Description:

> Expands a tag.

` div.hello#world`

will render:


```html
 <div class="hello" id="world"></div>```

` p hello world`

will render:


```html
 <p>hello world</p>```

Languages: ['html']



### a href

Description:

> Creates a link.

` a http://www.apptorium.com`

will render:


```html
 <a>http://www.apptorium.com</a>```

Languages: ['html']



### Template

Description:

> Basic HTML 5 template

` html`

will render:


```html
 <!DOCTYPE html>
 <html>
     <head>
         <title></title>
     </head>
     <body>
         
     </body>
 </html>```

Languages: ['html']
