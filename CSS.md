### Full Screen Background

Description:

> Creates a full screen background.

` fullbg ../assets/bg.jpg`

will render:


```css
 html {
     background: url("../assets/bg.jpg") no-repeat center center fixed;
     width: 100%;
     height: 100%;
 }
 
 
```

### Media Query

Description:

> Creates media query.

` mq 768`

will render:


```css
 @media only screen and (max-width: 768px) {
     
 }
```
